# 🎵 Music Web Player

A beautiful, modern music player built with vanilla HTML, CSS, and JavaScript. This application provides an elegant music listening experience with a clean UI, smooth animations, and responsive design.

## 🎥 Live Demo

🌐 **[Try the Live Demo](https://sagargupta16.github.io/Music-Web-App/)**

> **Note:** Make sure to enable audio in your browser and have speakers/headphones connected for the best experience!

## ⚡ Quick Start

1. **Clone & Run**
   ```bash
   git clone https://github.com/Sagargupta16/Music-Web-App.git
   cd Music-Web-App
   # Open index.html in your browser or serve locally
   python -m http.server 8000
   ```

2. **Open** `http://localhost:8000` in your browser

3. **Enjoy** your music with beautiful UI! 🎵

## 🎮 How to Use

1. **Launch the Player**
   - Open `index.html` in any modern web browser
   - The player will automatically load the first track

2. **Basic Controls**
   - Click the large **play/pause** button to start or stop music
   - Use **previous/next** buttons to navigate between tracks
   - Click and drag the **progress bar** to seek to any position

3. **Advanced Features**
   - Toggle **shuffle mode** to randomize track order
   - Enable **repeat mode** to loop the current track
   - Watch the album art **rotate** while music is playing

4. **Visual Feedback**
   - Active shuffle/repeat buttons are highlighted
   - Progress bar fills with gradient as song plays
   - Hover effects provide immediate visual feedback

## 🔧 Installation & Setup

### Quick Start

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Sagargupta16/Music-Web-App.git
   cd Music-Web-App
   ```

2. **Open in Browser**
   - Simply open `index.html` in your preferred browser
   - No server setup required - runs entirely in the browser
   - Or serve with a local server for development:

   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. **Add Your Music** (Optional)
   - Add your MP3 files to the `music/` folder
   - Add corresponding album art to the `images/` folder  
   - Update the `music_list` array in `script.js` with your tracks
